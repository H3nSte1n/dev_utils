# dev_utils

## Installation
### node
```sh
npm install devutils --save
```
### yarn
```sh
yarn add devutils
```

## Usage
### Javascript
```javascript
import { devUtils } from 'devutils';
```
### TypeScript
```typescript
import { devUtils } from 'devutils';
```
